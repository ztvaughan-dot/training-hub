Connecting Strava

Strava OAuth requires a registered app Client ID. Once connected, completed rides and runs sync automatically and populate real power, HR, and pace data.

  1. Go to strava.com/settings/api
  2. Create an app — name it anything (e.g. "Training Hub")
  3. Set Authorization Callback Domain to localhost for local use
  4. Copy your Client ID and paste below

Stored locally in your browser only — never sent anywhere except Strava's OAuth endpoint.

What syncs once connected: activity name, date, average power or pace, normalized power, average HR, TSS, and IF. Each activity is automatically matched to the closest scheduled workout.
